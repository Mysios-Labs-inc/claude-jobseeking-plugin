# Resume Content Analyzer

Comprehensive content analysis tool for strategic resume optimization. Evaluates space usage, identifies context gaps, and provides actionable recommendations before applying optimization workflows.

## 🎯 Core Analysis Functions

### 1. Content Density Analysis
**Evaluate space allocation across resume sections:**
- **Work Experience**: Analyze bullet density, achievement depth, quantification level
- **Projects**: Identify condensation opportunities (bullets → one-liners)
- **Skills**: Assess keyword coverage and strategic positioning
- **Other Sections**: Evaluate essential vs. supplementary content

### 2. Context Gap Identification
**Systematic review for missing high-value information:**
- **Travel/Remote Experience**: Cross-country work capability
- **Institutional Prestige**: Full school names, affiliations, programs
- **Project Purpose Clarity**: Specific user value beyond technical features
- **Business Impact**: Quantified outcomes and real-world applications
- **Industry Context**: Sector-specific achievements and domain expertise

### 3. Space Optimization Planning
**Strategic content allocation for 2-page PDF optimization:**
- **60% Work Experience**: Detailed achievements with X→Y→Z formula
- **25% Projects**: Condensed one-liners with key metrics preserved
- **10% Header/Summary/Skills**: Compact but professional
- **5% Education/Publications**: Essential information only

## 🔍 Analysis Categories

### Content Volume Assessment
**Space Usage Evaluation:**
- Word count and density per section
- Bullet point distribution analysis
- Information hierarchy assessment
- Redundancy identification

**Optimization Opportunity Ranking:**
1. **High Impact**: Projects with 3+ bullets (condensation candidates)
2. **Medium Impact**: Generic experience bullets (quantification opportunities)
3. **Low Impact**: Supplementary sections (space-saving potential)

### Context Completeness Review
**Missing Information Checklist:**
- [ ] Travel/remote work experience detailed
- [ ] Full institutional names and affiliations
- [ ] Project purposes and user impact clarity
- [ ] Quantified business outcomes
- [ ] Technical achievements with business context
- [ ] Industry-specific domain expertise
- [ ] Leadership and collaboration examples

### Strategic Improvement Opportunities
**Achievement Enhancement:**
- Experience bullets lacking metrics
- Generic technical descriptions
- Missing business impact connections
- Undervalued accomplishments

**Content Organization:**
- Section prioritization recommendations
- Information flow optimization
- Professional presentation improvements
