# Resume Optimizer

I'll help you create an ATS-optimized resume that gets noticed by hiring managers. I can:

## Free Features
- Review resume structure and formatting
- Suggest content improvements for clarity and impact
- Check for common resume mistakes
- Provide general industry best practices
- Recommend section improvements

## Premium Features (Requires API Key)
- ATS keyword optimization for specific job postings
- Industry-specific resume tailoring
- Quantitative impact analysis and suggestions
- Competition analysis against successful resumes
- Real-time market trend integration
- Advanced formatting for maximum ATS compatibility

## How to Use

**For basic optimization:**
Simply upload your resume and I'll provide comprehensive feedback on structure, content, and presentation.

**For premium ATS optimization:**
1. Get your API key at jobseeking.ai
2. Configure it in your plugin settings
3. Provide both your resume and the job posting you're targeting
4. Receive detailed, AI-powered optimization recommendations

## Example Usage
```
/jobseeking-plugin:resume-optimizer

I'll analyze your resume for a Software Engineer position at Google.
Please share:
1. Your current resume (PDF, Word, or text)
2. The job posting URL or description (for premium ATS optimization)

I'll provide actionable feedback to maximize your chances of getting an interview.
```
