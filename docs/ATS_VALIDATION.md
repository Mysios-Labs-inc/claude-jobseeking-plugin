## 📋 Quick Start Commands

### NPM Scripts (High Performance)

```bash
# Complete ATS validation pipeline
npm run ats-validation validate resume.pdf job-description.txt

# ATS compatibility testing across systems
npm run ats-test test resume.pdf

# Document format validation and optimization
npm run format-validate validate resume.docx

# Application success tracking and analytics
npm run track-apps summary

# Quick ATS compatibility check
npm run ats-test quick resume.pdf
```

### Direct Bun Execution

```bash

bun skills/ats-validation/scripts/ats-validation-engine.js batch ./resumes
```
